Описание плагина "Skfav. Коллективное избранное"
тема на livestreet.ru - http://livestreet.ru/blog/addons/14865.html

Реализует возможность просматривать ленту топиков (коллективных и персональных) отсортированных по колличеству добавлений в избранное.

Плагин является дополнительным инструментом в поиске нужной и полезной информации для любого сайта, работающего на Livestreet.

По умолчанию адрес страницы с лентой http://site.ru/favourite_top/
